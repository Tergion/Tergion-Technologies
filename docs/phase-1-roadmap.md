# Phase 1 Roadmap

## Phase 1A

Foundation, routing, design system, legal drafts, lead form, server-side lead route, security stubs, provider stubs, and build verification.

## Phase 1B

Configure production credentials, live Turnstile, live email, Google Sheets or CRM destination, production rate limiting, analytics decision, legal review updates, and visual QA.

## Phase 1C

Refine content, add approved founder/company story, add real implementation proof when available, expand example pages, and tune conversion paths based on actual usage.

## Deferred

No chatbot, accounts, payments, dashboard, client portal, fake testimonials, fake reviews, unsupported statistics, or live GoHighLevel credential integration in Phase 1A.
